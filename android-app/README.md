# L4 App 🥒

A simple Android demo app built by L4 (Leon AI 4).

## What it does

- Shows a "Hello from L4 🥒" message
- Has a tap counter button
- Looks slick with a dark theme

## Download & Install

> **The APK is tracked in this repo** — committed at `android-app/app-debug.apk`

### Option 1: Download from GitHub (recommended)

1. Open the repo on GitHub: https://github.com/Walusimbi-Leon1/Codespace-3
2. Navigate to `android-app/` → `app-debug.apk`
3. Click the **Download** button (or open the file and click the download icon)
4. On your Android phone:
   - Open your Downloads or Files app
   - Tap `app-debug.apk`
   - You'll see a warning about installing from unknown sources — tap **Install anyway** or **Settings → Allow from this source**
   - Tap **Install**

### Option 2: Direct link

[https://github.com/Walusimbi-Leon1/Codespace-3/raw/main/android-app/app-debug.apk](https://github.com/Walusimbi-Leon1/Codespace-3/raw/main/android-app/app-debug.apk)

Open this link on your phone to download directly.

### Requirements

- Android 7.0 (API 24) or higher
- "Install from unknown sources" enabled (temporary — you can disable after installing)

## Building yourself

```bash
# Clone the repo
git clone https://github.com/Walusimbi-Leon1/Codespace-3.git
cd Codespace-3/android-app

# Install SDK (requires Java 17+ and Android SDK 34)
export ANDROID_HOME=/path/to/android-sdk
./gradlew assembleDebug

# APK at: app/build/outputs/apk/debug/app-debug.apk
```
